1. ```git clone https://github.com/LxKxC/LxKxC.github.io.git #``` 将仓库克隆 到本地

2. 进入到本地仓库文件夹 将想要删除的文件删除

3. 右键 ```git bash here```

4. ```git add .```

5. ```git commit -m "say some thing "```  双引号中内容按照自己需求填写

6. ```git push -u origin master```
